Inserindo vídeos hospedados localmente

Este vídeo está hospedado no meu próprio servidor.

Forma mais simples

  1. Tag VIDEO
  2. Dentro da tag em SRC coloca o nome do vídeo
  3. Coloca o WIDTH desejado
  4. Acrescenta CONTROLS
  5. Coloque MUTED antes do AUTOPLAY para o vídeo começar automaticamente

Forma mais ampla

  1. Tag VIDEO
  2. Apaga a SRC e dentro da tag digita SRC:SZ
  3. Preenche as informações
  4. Colocar o WIDTH desejado
  5. Acrescentar na Tag: MUTED; AUTOPLAY; LOOP

Inserindo Vídeos Externos

  1. Escolha um vídeo no YouTube
  2. Clique em compartilhar
  3. Depois em incorporar vídeo
  4. Clicar na opção copiar
  5. Colar no Visual Code

OBS.: Outra opção para vídeos externos é o Vimeo.
