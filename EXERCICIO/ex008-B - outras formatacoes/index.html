Código-fonte

Para adicionar um código no site utilize a tag CODE.

Exemplo: O comando document.getElementById('teste') é utilizado na linguagem HTML.

Para formatar o código no estilo padrão dentro do site coloque a tag PRE antes de CODE. Caso queira move-lo selecione todo o código e:

SHIFT + TAB move para esquerda

TAB move para direita

        
    num = int(input('Digite um número'))

    else:

    print('Fim do programa')

Citação simples

Utilizar a tag Q de quote em inglês para adicionar as aspas.

Exemplo: Como diria o pai de um amigo: O computador é um burro muito rápido.

Citação completa

A tag BLOCKQUOTE para inserir a citação direta. E dentro da tag colocar CITE caso queira adicionar o link para o documento.
