Usando links externos

Para inserir um link externo digite a tag A e preencha o HREF com a URL do site, por exemplo, "https:/google.com/".

Para que ao clicar no link seja direcionado para uma nova aba insira dentro da tag A:

  • target="_blank"
  • rel="external"

Usando links internos

Com o atalho CTRL + barra de espaço escolher qual arquivo.html você deseja adicionar.

Esta é a primeira página do site. Caso queira, você pode acessar a minha segunda página.

Para informar que o link será a próxima página digite: rel="next"

Para informar que o link será a página anterior digite: rel="prev"

Para link de patrocínio é recomendado: rel="nofollow"


Observação!

Para abrir o link na própria aba do site digite dentro da tag A:

target="_self"


Para informar que o arquivo está em uma pasta anterior digite tag a href="../nomedoarquivo.html".

Para cada sequência de ../nomedoarquivo.html o código volta uma página.
